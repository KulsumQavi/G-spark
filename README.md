# 🌾 FarmLinks — Smart Crop Transport Coordination System

> An AI-powered, IoT-enabled, 3-sided marketplace connecting **Farmers**, **Transporters**, and **Buyers** for efficient crop logistics in rural India.

---

## 🎯 Problem Statement

India loses **₹92,000 Crore** every year due to post-harvest logistics failure. Existing crop transport platforms fail because of:

1. ❌ Poor coordination between multiple pickups
2. ❌ No real-time tracking of vehicles
3. ❌ Drivers prioritized higher-paying routes (small farmers ignored)
4. ❌ Demand spikes during harvest seasons crashed the system
5. ❌ Lack of reliability in scheduling

## 💡 Our Solution

**FarmLinks** is a smart 3-sided platform that solves all 5 gaps:

| Problem | Our Solution |
|---------|-------------|
| Coordination | AI Clustering of nearby farmers (60% cost reduction) |
| No Tracking | Live GPS + IoT temperature sensors + SMS alerts |
| Bad Incentives | Points & Tier System (Bronze → Diamond) |
| Harvest Overload | AI Demand Forecasting |
| No Reliability | Two-way ratings + accountability |

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** Python (Flask)
- **Database:** SQLite
- **AI/ML:** scikit-learn, Google OR-Tools (route optimization)
- **IoT:** GPS simulation, Temperature sensor monitoring
- **Notifications:** SMS gateway integration

---

## 🚀 Features

### 👨‍🌾 For Farmers
- Submit transport requests with crop details
- Live truck tracking via GPS
- SMS notifications at every step
- Rate drivers after delivery

### 🚛 For Transporters
- Get AI-optimized multi-stop routes
- Earn points & badges for reliability
- Auto UPI payments after delivery
- Tier-based bonus pay

### 🏪 For Buyers
- Browse fresh crops in marketplace
- Pre-order from upcoming harvests
- Live ETA tracking
- Quality data via IoT sensors
